##  Overview

**Egators** is a fully responsive, multi-page educational website built from scratch using pure HTML, CSS, and JavaScript. It includes a smooth navigation bar, course listings, interactive FAQs, a testimonials slider, team section, and a contact form — all optimized for desktop, tablet, and mobile screens.

---

##  Pages

| Page | Description |
|:---|:---|
| `index.html` | Home — hero section, categories, popular courses, FAQs, testimonials |
| `about.html` | About — achievements stats, meet the team section |
| `course.html` | Courses — full course listing grid |
| `contact.html` | Contact — contact form with sidebar details |

---

##  Features

| Feature | Description |
|:---|:---|
|  Fully Responsive | Optimized for desktop, tablet & mobile |
|  Fixed Navbar | Scroll-aware navbar with animated mobile menu |
|  Course Grid | Responsive 3-column course card layout |
|  Interactive FAQs | Toggle open/close with icon animation |
|  Testimonials Slider | Swiper.js carousel with pagination |
|  Achievements Section | Stats cards with hover effects |
|  Team Section | Hover-reveal social links with grayscale image effect |
|  Contact Form | Aside + form layout with social links |
|  CSS Variables | Consistent theming via `:root` variables |

##  Tech Stack

| Tool | Purpose |
|:---|:---|
| HTML5 | Page structure & semantic markup |
| CSS3 | Styling, grid layout, animations, media queries |
| JavaScript | Navbar toggle, FAQ accordion, scroll effects |
| Swiper.js | Testimonials carousel slider |
| Unicons | Icon library |
| Google Fonts | Montserrat font family |

## How to Run

**1. Clone the repository**

```bash
git clone https://github.com/your-username/egators-educational-website.git
cd egators-educational-website
```

**2. Open in browser**

Simply open `index.html` in your browser — no build tools or dependencies needed.

```bash
# Or use VS Code Live Server
# Right click index.html → Open with Live Server
```

>  No npm, no frameworks, no setup — pure HTML/CSS/JS.
